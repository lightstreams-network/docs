# Lightstreams Wiki

## Requirements

- Ruby Version Manager (RVM)
- Ruby 2.3.3
- Jekyll


1. Follow instructions for installing RVM
https://rvm.io/

2. Install Ruby 2.3.3
```
$ rvm install 2.3.3
```

3. Install Jekyll Gem
```
gem install jekyll bundler
```

## Running locally

1. Run Jekyll
```
$ bundle exec jekyll serve
```

2. browse to http://localhost:4000
